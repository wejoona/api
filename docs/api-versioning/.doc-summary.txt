-------------------------
1. Amount format: Float dollars → Integer cents

3. Error format: String messages → Structured codes
4. Pagination: Offset-based → Page-based
5. Balance format: Single currency → Multi-currency
6. Transfer request: Flat object → Nested recipient
7. Idempotency support (new)
8. Request tracing (new)
9. Rate limit headers (new)
10. Enhanced webhook payloads
