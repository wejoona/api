================================================================================
NEW API ENDPOINTS
================================================================================

1. GET /admin/dashboard
   Auth: JWT + Admin role
   Cache: 1 minute
   Response: Basic dashboard statistics
   Example:
   {
     "totalUsers": 1250,
     "totalTransactions": 5420,
     "totalVolume": 125000.50,
     "todayVolume": 2340.25,
     ...
   }

2. GET /admin/dashboard/enhanced?days=30
   Auth: JWT + Admin role
   Cache: 1 minute
   Query Params:
   - days: 1-365 (default: 30)
   Response: Enhanced dashboard with time-series data
   Example:
   {
     "totalUsers": 1250,
     "totalTransactions": 5420,
     "transactionTimeSeries": [...],
     "userGrowthTimeSeries": [...],
     "transactionsByType": {...},
     "transactionsByStatus": {...}
   }

3. POST /admin/dashboard/cache/invalidate
   Auth: JWT + Admin role
   Response: { "message": "Dashboard cache invalidated" }

4. GET /admin/reports/user-growth-timeseries?days=30
   Auth: JWT + Admin role
   Query Params:
   - days: 1-365 (default: 30)
   Response: User growth time-series data
